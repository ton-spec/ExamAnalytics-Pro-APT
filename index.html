Installation

Install on Ubuntu, Linux Mint or Debian:

  1. Download the package: Download .deb (v1.0.1)
  2. Double-click the downloaded file to install it with your system's package installer.
  3. Or install from a terminal:
    sudo dpkg -i examanalyticspro_1.0.1_amd64.deb
  4. On first launch, the app displays a unique Client ID for your device.
  5. Copy the Client ID and email it to examanalyticsadmin@gmail.com to request your activation token.
  6. Once you receive your activation token, paste it into the app's activation screen and click Verify_Activate.
